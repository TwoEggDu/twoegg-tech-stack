```text
2. 加载加速：商业版优化了哪些步骤  - 职责：metadata 解析惰性化 / 批量注册 / 预计算索引等可能的手段
3. metadata 内存占用：社区版的内存构成  - 职责：InterpreterImage 持有什么、type/method/field 定义各占多少
4. metadata 内存优化：商业版减了哪部分  - 职责：共享结构 / 延迟分配 / 压缩存储等可能的手段
5. Incremental GC：社区版为什么不支持  - 职责：解释器栈帧和对象引用在 GC write barrier 层面的冲突
6. Incremental GC 适配：商业版改了什么  - 职责：解释器侧的 write barrier 插入或 GC root 注册方式的变更
7. 三者的工程影响：加载时间、峰值内存、GC 卡顿分别改善多少  - 职责：用可测量指标收束
```
